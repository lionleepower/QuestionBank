# QuestionBank

这是我的个人学习问题题库，用来记录学习过程中遇到的问题、自己的理解、批改反馈，以及最后需要记住的结论。

题库的目标不是只保存答案，而是把“我当时怎么想、哪里理解错了、最后应该怎么记”都留下来。之后复习时，可以通过随机抽题的方式检查自己是否真的掌握。

## 文件结构

当前题库按主题拆分为多个 Markdown 文件，例如：

- `01_bash.md`：Bash、shell、Slurm、benchmark 脚本相关问题。
- `02_python.md`：Python 或后续 Python 学习相关问题。

以后可以继续按主题新增文件，例如：

```text
03_git.md
04_machine_learning.md
05_linux.md
```

## 记录格式

每一道题使用二级标题作为边界，格式为：

```markdown
## YYYY-MM-DD：问题标题

**我的问题**

...

**我的理解 / 回答**

...

**批改**

...

**记住**

...
```

其中：

- `我的问题`：记录当时问的问题。
- `我的理解 / 回答`：记录自己先尝试回答的内容。
- `批改`：记录纠正、补充和解释。
- `记住`：压缩成以后复习时最重要的结论。

## 随机抽题复习

可以使用 `scripts/quiz.py` 从所有题库 Markdown 文件中随机抽题。默认会抽 10 道题，只显示题目和问题内容，隐藏 `批改` 和 `记住`，方便先自己回答。

```bash
python3 scripts/quiz.py
```

也可以指定抽题数量，范围是 10 到 20 道：

```bash
python3 scripts/quiz.py 20
```

如果输入的数量小于 10，会按 10 道处理；如果大于 20，会按 20 道处理。如果当前题库里的题目总数少于请求数量，脚本会展示全部题目。

## 推荐复习方法

1. 运行随机抽题脚本。
2. 先不要看原文件里的 `批改` 和 `记住`。
3. 用自己的话回答每一道题。
4. 回到原题库文件，对照 `批改` 和 `记住` 检查。
5. 如果答错或答不完整，就补充自己的理解，或者新增一条更具体的问题。
